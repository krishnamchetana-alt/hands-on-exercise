# ReactJS Hands-on Project

A beginner-friendly React task tracker demonstrating functional components, `useState`, `useMemo`, event handlers, controlled inputs, conditional rendering, and list rendering.

## Run locally

1. Install Node.js 18 or newer.
2. Run `npm install`.
3. Run `npm run dev` and open the local URL shown by Vite.

## Build

Run `npm run build` to create a production build in `dist/`.
